# Agentic AI Projects

This repository contains a collection of Agentic AI projects built using modern LLM frameworks and automation tools.  
The projects demonstrate autonomous AI agents capable of research, planning, content generation, and problem solving.  
Technologies used include LangGraph, Microsoft AutoGen, Python, and other AI orchestration frameworks.  
Each project showcases real-world use cases such as AI chatbots, blog generation, vacation planning, and automated DSA solving.  
The goal of this repository is to demonstrate practical implementations of agentic AI systems and intelligent workflows.
